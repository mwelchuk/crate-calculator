Crate Calculator

This page attempts to work out measurements for creating a crate using wood dimensions provided. 2 cuts of wood are needed to create these crates, a small rectangular lengths for the verticals and bottom supports, thin wide lengths for the sides and bottoms. The original was built using planed approx 16x36mm timber and feather edge boarding (approx 10x97mm).

Design

The design holds 20 bottles of the size choosen. I have found it useful to size the crates I've made to fit the largest combined diameter/height of the bottles I have. In fact, this allows me to cram an extra row of bottles in when I use the crates with the taller, thiner bottles I have, so the crates hold 24 bottles for me.
